# TOP Todo List

A simple, responsive Todo List app built for The Odin Project. It uses Webpack, vanilla JS, and modular components. Create projects, add tasks, and manage them with a clean UI.

## Features

-   Project management with active project highlighting
-   Create tasks with title, description, due date, priority, and project
-   Task cards with priority color accents and delete action
-   Accessible focus states and improved mobile responsiveness
-   Webpack-powered build with HTML/CSS handling

## Tech Stack

-   JavaScript (ES Modules)
-   Webpack 5 + Dev Server
-   HTML/CSS (vanilla)

## Usage Tips

-   Use the “Create Project” button to add a project; it appears in the sidebar and in the task modal project dropdown.
-   Click a project to set it active; the button highlights accordingly.
-   Open “Create Task” to add a task to a chosen project.

## Scripts

-   `npm run dev` – start dev server
-   `npm run build` – build production bundle
-   `npm run deploy` – push `dist/` to `gh-pages` via subtree (requires `origin` and branch setup)

## Notes

-   The app currently stores data in local storage.
